# GhostPipe
GhostPipe — скоростной асинхронный скрапер и чекер прокси на Python. Использует httpx и asyncio для параллельной проверки сотен узлов. Функции: автоматический сбор из источников, замер latency, фильтрация нерабочих IP и экспорт в .txt. Легкий, модульный и эффективный инструмент для обеспечения анонимности и автоматизации.
